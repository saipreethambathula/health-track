# HealthTrack Dashboard

A responsive and user-friendly **health tracking dashboard** built with **HTML and CSS**. It helps patients manage health metrics, appointments, prescriptions, and profile details in a clean interface.

---

## Features

* **Dashboard Overview**: Quick glance at key health metrics like heart rate, blood pressure, oxygen level, and BMI.
* **Appointments Management**: View upcoming appointments with doctors and their status.
* **Prescription Tracking**: List of prescribed medicines with dosage and duration.
* **Patient Profile**: View detailed patient information including contact and doctor assigned.
* **Responsive Layout**: Fully responsive and works on desktop, tablet, and mobile devices.

---

## Tech Stack

* **Frontend:** HTML, CSS
* **Design:** Responsive dashboard layout using CSS Grid and Flexbox
* **Icons:** Font Awesome
